This is another test file. It can be used to document the feature that allows for multiple documents to be opened from the same instance of WebWhirr.

To use this, either press Ctrl/Command-O, or click on the file menu and select the "Open" option. Select one of the test files included with WebWhirr--that is, test.html, test2.html, test3.html, or test4.html, or any other HTML files that may be on your computer.

Eventually, WebWhirr will gain more features, such as:

  • the ability to connect to the internet
  • a fully featured, customizable interface

For now, however, the focus is on building the rendering base that will be used in the browser.
